## Active Tasks
Task context lives in `.ai/tasks/[TASK-ID]/`. When working on a Jira ticket, load:
- `context.md` — background and key findings
- `checklist.md` — implementation steps
- `findings.md` — gotchas and discoveries
- `tracker.md` — status and PR links
- `task-plan.md` — design and phase breakdown

See `AGENTS.md` for full resource registry, conditional context files, and skills registry.
